## 最低限の前提条件

- Git / Node.js / npm が利用可能
- Googleアカウントと GitHub アカウントが利用可能
- リポジトリへのアクセス権が付与済み

## 完了条件（入口確認）

以下を満たしたら環境構築は完了です。

- `GAS_scripts` に `.clasp.json` と `appsscript.json` が存在する
- scriptId がチーム内で共有されている
- `develop` と自分の `feature/*` ブランチで作業できる
- ブランチ保護ルールにより `main` / `develop` へ直接 push できない

## 補足

- ディレクトリ名は実体に合わせて `GAS_scripts` を使用する
- 追加情報は [docs索引](docs索引.md) から辿る
