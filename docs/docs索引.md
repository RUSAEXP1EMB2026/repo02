# docs索引

## 目的

docs配下の文書を「どの順で」「どれを正本として」参照すべきかを明確化し、重複参照と運用の迷いを防ぐ。

## 対象読者

- プロジェクト参加メンバー全員
- チームリーダー（GASプロジェクト作成担当）
- レビュー担当（仕様/設計/運用の確認者）
