# 要求仕様書：電子レンジとNature Remoの連携による操作支援システム

## 1. 全体概要

### 1.1 システムの概要
電子レンジの扉の開閉をトリガーとし、スマートスピーカーによる直感的な音声ガイダンス、保護者からのLINEを通じた遠隔承認、そしてNature Remoと物理操作デバイス（指ロボット）を連携させることで、子供や高齢者が安全かつ迷わず電子レンジを使用できる環境を提供する。操作状況は保護者のスマートフォンにリアルタイムで通知され、必要に応じて遠隔からの操作指示も可能とする。

### 1.2 製品の機能
- **直感的な音声ガイダンス**: 電子レンジの扉開閉や操作状況に応じて、スマートスピーカーからステップバイステップの音声案内を行う。
- **保護者による遠隔承認・操作**: 子供や高齢者が電子レンジの準備を完了した際、保護者のLINEに通知を送り、保護者がLINEメッセージで加熱の承認や加熱モード（例: 「ご飯」「おかず」）を選択・指示できるようにする。
- **物理ボタン操作**: Nature Remo APIを介して指ロボットを制御し、電子レンジの「あたため」ボタンや「スタート」ボタン、または特定の加熱モードボタンを物理的に押す。
- **利用状況の記録と通知**: 電子レンジの利用開始・終了時刻、選択された加熱モードをGoogleスプレッドシートに記録し、LINE Notifyを通じて保護者に詳細な状況を通知する。
- **異常検知・安全対策**: 加熱終了後、一定時間扉が開かれない場合の取り忘れ防止警告、加熱中の扉開放検知による停止など、安全性を高める機能を提供する。

### 1.3 想定する利用者の特性
- **子供（小学生以下）**: 一人で留守番中に、火傷や設定ミスを恐れずに安全に食事を温めたい子供。音声ガイダンスと保護者の承認により、安心して操作できる。
- **高齢者**: 複雑なボタン操作や液晶画面の確認が難しく、日常的な家電操作に不安を感じている高齢者。音声ガイダンスと簡略化された操作フローにより、迷わず使用できる。
- **保護者・介護者**: 外出先から子供や高齢者の家電利用状況を把握し、安全を見守りたい人。遠隔承認機能により、安全性を確保しつつ、必要な時にサポートできる。

## 2. 詳細仕様

### 2.1 機能要求
- **扉開閉検知と音声ガイダンス**: 
    - 磁気センサーが扉の「開」を検知した際、3秒以内にスマートスピーカーから「お皿を入れて扉を閉めてください。温める準備ができたら、ママ（パパ）にLINEで『温めて』と送ってね」という音声を再生すること。
    - 扉が「閉」の状態に戻った際、5秒以内にスマートスピーカーから「扉が閉まりました。ママ（パパ）からの指示を待っています」という音声を再生すること。
- **保護者への通知と承認リクエスト**: 
    - 扉が閉まったことを検知した後、保護者のLINEに「〇〇ちゃんが電子レンジの準備をしました。温めますか？」というメッセージと共に、「ご飯を温める」「おかずを温める」「キャンセル」の選択肢を送信すること。
- **遠隔加熱実行**: 
    - 保護者がLINEで「ご飯を温める」を選択した場合、指ロボットが電子レンジの「ご飯あたため」ボタン（または「あたため」ボタンを複数回押すなど、事前に設定されたシーケンス）を物理的に押すこと。
    - 保護者がLINEで「おかずを温める」を選択した場合、指ロボットが電子レンジの「おかずあたため」ボタン（または同様のシーケンス）を物理的に押すこと。
- **ステータス通知**: 
    - 加熱開始時に、LINE Notifyで「電子レンジの使用を開始しました（〇〇モード）」と通知すること。
    - 加熱終了（電力センサーまたは指ロボットの動作完了）時に、スマートスピーカーから「温めが終わりました。熱いから気をつけて取り出してね」と音声を再生し、LINE Notifyで「温めが終了しました」と通知すること。
- **ログ記録**: 
    - 利用日時、操作内容（加熱開始/終了）、選択された加熱モード、その時の室温をGoogleスプレッドシートに1行ずつ追記すること。
- **取り忘れ防止機能**: 加熱終了から10分経過しても扉が開かれない場合、スマートスピーカーで「レンジの中に食べ物が入っています」と通知し、保護者のLINEにも通知すること。
- **加熱中の扉開放検知**: 加熱中に扉が開かれた場合、直ちに指ロボットの動作を停止させ、スマートスピーカーで「扉が開きました。加熱を中断します」と通知すること。

### 2.2 非機能要求
- **応答速度**: センサー検知から音声ガイダンス開始、および保護者のLINE承認から指ロボットの動作開始までの遅延時間は、ネットワーク環境を含めて5秒以内であること。
- **信頼性**: システムの稼働状況を24時間監視し、Nature Remoのオフライン状態が1時間以上続いた場合に管理者にメールで通知すること。
- **操作性**: ユーザー（子供・高齢者）は電子レンジの扉開閉と、必要に応じて保護者へのLINEメッセージ送信のみで加熱操作が完了すること。保護者はLINEアプリから直感的に承認・操作モードを選択できること。
- **安全性**: 加熱中の扉開放検知による自動停止、取り忘れ防止警告により、火傷や事故のリスクを最小限に抑えること。

## 3. 使用デバイス・サービス一覧

| 項目 | 具体的なデバイス・サービス | 役割 |
| :--- | :--- | :--- |
| **スマートリモコン** | Nature Remo 3 | 指ロボットの制御、温度監視、ハブ機能。 |
| **物理操作デバイス** | SESAME Bot 2 / SwitchBot ボット | 電子レンジのボタンを物理的に押す。 |
| **開閉検知センサー** | 磁気開閉センサー | 電子レンジの扉の開閉を検知する。 |
| **制御ロジック** | Google Apps Script (GAS) / AWS Lambda | センサー入力を受け取り、APIを叩くロジック、LINEメッセージの送受信。 |
| **通知・承認サービス** | LINE Messaging API / LINE Notify | 保護者へのリアルタイム通知、遠隔承認・操作。 |
| **記録サービス** | Google スプレッドシート | 利用ログの保存。 |
| **音声案内** | Google Home / Amazon Alexa | ユーザーへの操作ガイダンス、警告。 |
